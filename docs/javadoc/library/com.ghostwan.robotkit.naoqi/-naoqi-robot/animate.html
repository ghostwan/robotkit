animate

suspend fun animate(@RawRes mainAnimation: Int, @RawRes vararg additionalAnimations: Int, throwOnStop: Boolean = true, onStart: suspend () -> Unit = null, onResult: suspend (Result<Void>) -> Unit = null): Unit

Animate the robot with an animation using raw qianim file

If onResult it's not set, the coroutine will be suspended until the animation end, without blocking the thread

Parameters

mainAnimation - Main animation. It's an android raw resources.

additionalAnimations - Additional animation as for example trajectory to make the robot dance. They are android raw resources.

throwOnStop - By default it's true. Does this method throw when the stop api is called and stop it. Could be useful to set it to false when we want to continue actions scenario after a stop

onStart - This lambada is called when the action start on the robot

onResult - If this lambada is set the coroutine won't wait and the result of the api call will be forward to the lambda. Give a Success or a Failure with the exception
