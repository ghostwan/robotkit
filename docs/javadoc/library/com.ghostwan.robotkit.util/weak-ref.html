weakRef

inline fun <T> weakRef(value: T): WeakRefHolder<T>

Use the by keyword to delegate weak references.

Internally this creates a WeakRefHolder that will store the real WeakReference. Example usage:

    var weakContext: Context? by weakRef(null)
    …
    weakContext = strongContext
    …
    context = weakContext
